Project Description

Library App used to track books you are reading or have read.

Project Details

  • Toggle dark/light mode
  • Detects user's dark/light mode preference from their OS settings
  • Add books, track read status, or remove books from library
  • Fetches data from OpenLibrary API to populate title & cover image
